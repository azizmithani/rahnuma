Rahnuma: Pathway Prediction and Network Analysis Tool

Help ‒ Parameters Page


Parameters

  Organism(s) Allows the user to select either of the following two options.
  1. Reference: Builds the metabolic network using KEGG's reference pathway.
  2. Select Organism(s): Builds the metabolic network using one or more organism(s) selected by the user. Multiple organisms may be selected by holding the ‘Ctrl’ key. The list of organisms available in Rahnuma can be found here.
  Mode Available only if “Collective” Network Mode is selected. Metabolic network can be built in two modes.
  1. All Reactions: Uses all reactions present in at least one of the organisms to build the metabolic network.
  2. Common Reactions: Uses only those reactions that are common to all organisms to build the metabolic network.
  3. Reaction Neighbourhood: Considers reactions which meet the neighbourhood cutoff, i.e. the proportion of neighbours present in at least one of the networks is greater than the specified cutoff.

Note: The interpretation of output for different combinations of organism mode for comparative analysis can be found here.

  Phylogeny The phylogeny may be specified in a format based on Newick format where each subtree must have exactly two children. For example, (A,B), ((A,B),C), ((A,B),(C,D)) etc. are considered as valid phylogenies whereas phylogenies like (), (A), and (A,B,C) are regarded as invalid. Each entry in the specified phylogeny must correspond to a valid organism code in KEGG. The list of organisms available in Rahnuma can be found here.

Note: White spaces are ignored from the phylogeny so a phylogeny can be specified over multiple lines or formatted for typographical convenience.

Network Building Parameters

  Neighbourhood Cutoff Cutoff value for proportion of neighbour reactions to be present in the ancestral/combined network for a reaction. Reactions with neighbourhood proportion value less than specified cutoff are not included in the ancestral/combined network. Cutoff value must be between 0.0 and 1.0 exclusive.
  Ignore Metabolites (Optional) A list containing metabolites Ids from KEGG to be avoided when counting neighbourhood reactions. Reactions connected through these metabolites are ignored. File must be tab separated with each line in the following format.

  1. Id (a valid compound id from KEGG)
  2. Description (Optional)
A list of typical metabolites to be ignored can be downloaded here.

Pathway Specific Parameters

  Start / End Metabolites Set of metabolites to be used as start and end metabolites. One or more metabolites may be selected for each set such that there is no metabolite common to both sets. Multiple metabolites may be selected by holding the ‘Ctrl’ key. The Ids shown in brackets (CXXXXX) are corresponding compound Ids in KEGG.

Subset: Filters metabolites for easy search and selection.
  Pathway Prediction Mode Pathway prediction can be performed in two modes.
  1. Connection: Works on predefined connections between metabolites in a reaction.
  2. Reaction: Uses all possible connections between the metabolites by pairing a substrate with all product metabolites. (Warning: Not recommended for high cutoff lengths!)
Note: Reactions with no enzymes associated or those with general chemical equations are ignored during pathway prediction. Connections passing through metabolites with no structural information are also ignored.
  Connection Option The user can choose from the following connection options.
  1. Carbon connections: Manually curated connections tracing the movement of one or more carbon atoms between metabolites.
  2. Nitrogen connections: Manually curated connections tracing the movement of one or more nitrogen atoms between metabolites.
  3. KEGG RPairs: Connection defined by KEGG's Reactant Pairs (RPairs).
Note: Manually curated carbon and nitrogen connections are available only when curated pathways are used to build metabolic network(s).
  Optional Constraint(s) Two files may be optionally uploaded to the server.
  1. Ignore Metabolites: A list containing metabolites Ids from KEGG to be avoided during pathway prediction. Pathways passing through these metabolites are not reported.
  2. Required Elements: A list of elements that must be present in all the compounds occurring in a pathway.

Files must be tab separated with each line in the following format.

  1. Id (a valid compound id from KEGG or element symbol)
  2. Description (Optional)
A list of typical metabolites to be ignored can be downloaded here.
  Cutoff Length Maximum length of a pathway. Currently, pathways with length upto 10 reactions can be predicted. Pathways greater than the cutoff length are not reported. For cutoff length greater than 6, the default list of ignore metabolites is automatically included. The list of metabolites ignored is available here.
  Return Value Only required for pathway prediction option or if pathway based comparative analysis is selected. Two return values are possible.
  1. Pathways: Reports complete pathways.
  2. Reactions: Returns only the reactions involved in the pathways.
