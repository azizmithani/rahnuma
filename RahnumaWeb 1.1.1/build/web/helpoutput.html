Interpretation of Organism Mode for comparative analysis:

For comparative analysis, three sets of results are computed:

  1. Reactions/Pathways present in the network built using Organism Set 1 that are absent from the network built using Organism Set 2.

  2. Reactions/Pathways present in the network built using Organism Set 2 that are absent from the network built using Organism Set 1.

  3. Reactions/Pathways present in the network built using Organism Set 1 as well as the network built using Organism Set 2.

The values in each result set depends on the Organism Mode selected when submitting a job. These are explained below.

Organism Set 1

All Reactions
  1. Reactions/Pathways present in at least one organisms in "Organism Set 1" that are not present in any organism in "Organism Set 2".
     
  2. Reactions/Pathways present in at least one organism in "Organism Set 2" that are not present in any organism in "Organism Set 1".
     
  3. Reactions/Pathways present in at least one organism in both "Organism Set 1" and "Organism Set 2".
  1. Reactions/Pathways present in one or more organisms in "Organism Set 1" that are not present in all organisms in "Organism Set 2", but may be present in one or more organisms.
     
  2. Reactions/Pathways present in all organisms in "Organism Set 2" that are not present in any organism in "Organism Set 1".
     
  3. Reactions/Pathways present in all organisms in "Organism Set 2" and at least one organism in "Organism Set 1".
Common Reactions
  1. Reactions/Pathways present in all organisms in "Organism Set 1" that are not present in any organism in "Organism Set 2".
     
  2. Reactions/Pathways present in at least one organism in "Organism Set 2" that are not present in all organisms in "Organism Set 1".
     
  3. Reactions/Pathways present in all organisms in "Organism Set 1" and at least one organism in "Organism Set 2".
  1. Reactions/Pathways present in all organisms in "Organism Set 1" that are not present in all organisms in "Organism Set 2".
     
  2. Reactions/Pathways present in all organisms in "Organism Set 2" that are not present in all organisms in "Organism Set 1".
     
  3. Reactions/Pathways present in all organisms in both "Organism Set 1" and "Organism Set 2".
