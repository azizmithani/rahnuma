Rahnuma: Pathway Prediction and Network Analysis Tool

Help ‒ Main Page


Job Detail

  Job Name Free text field to specify a meaningful name for the job.
  Email A valid email address. A notification is sent to the user when results are ready.


Dataset

  KEGG Pathways Allows the user to select one of the following options.
  1. All Available KEGG Pathways: Builds metabolic network(s) using all KEGG pathway maps involved in metabolism.
  2. All Curated KEGG Pathways: Builds metabolic network(s) using pathway maps which have been curated for carbon and nitrogen connections. The list of curated pathway maps can be found here.
  3. Select Pathway Map(s): Builds metabolic network(s) using one or more pathway map(s) selected by the user. Multiple pathway maps may be selected by holding the ‘Ctrl’ key.

Job Specification

  Analysis Rahnuma consists of three modules:
  1. Pathway Analysis
  2. Comparative Analysis
  3. Network Analysis

Select desired module to view available options.
  Analysis Type Pathway Analysis
  1. Pathway Prediction: Allows the user to perform simple pathway prediction between two or more metabolites.
  2. Reactions Exclusive to a Pathway: Identifies reactions that are involved in pathways from only one of the specified start metabolites.
  3. Reactions Acting as Bridges in a Pathway: Identifies reactions which if deleted from the network will result in all pathways being removed between the specified metabolites.

Comparative Analysis

  1. Standard Comparison: Performs comparative analysis over a phylogeny or between two groups of organisms.
  2. All But One Comparison: Identifies pathways or reactions that are present in only one organism in a group but absent in all other organisms and vice versa.

Network Analysis

  1. Ancestral Network Building: Builds metabolic networks over a phylogeny and reports the reactions present at various levels of the phylogeny
  Analysis Mode Pathway Analysis / Comparative Analysis*
  1. Organism Mode: Allows the user to select the reference network built using KEGG's reference pathway, or to specify one or more organisms.
  2. Phylogeny Mode: Asks the user to specify a phylogeny over which analysis must be performed.

Network Analysis

  1. All Reactions: Combines reactions from all the networks below the current level of the phylogeny.
  2. Common Reactions: Considers only those reactions that are present in all the networks present below the current level.
  3. Reaction Neighbourhood: Considers reactions which meet the neighbourhood cutoff, i.e. the proportion of neighbours present in at least one of the two networks below the current level is greater than the specified cutoff.
  4. Parsimony: Builds networks by maximum parsimony approach using the Fitch algorithm (Fitch 1971) with ties resolved in the favor of presence of a reaction.

* Phylogeny Mode is not available for “All But One Comparison”.
  Comparison Mode Comparative Analysis is available in two modes:
  1. Full Network: Takes into account all reactions present in the networks.
  2. Pathway Based: Performs comparison based on reactions involved in predicted pathways between two or more metabolites.
  Network Mode Available only for Pathway Analysis and Phylogeny based Comparative Analysis.

Organism Mode

  1. Individual: Separate network is built for each organism (in case more than one organism is selected).
  2. Collective: One single network is constructed using reactions from the selected organisms.

Phylogeny Mode

  1. All Reactions: Combines reactions from all the networks below the current level of the phylogeny.
  2. Common Reactions: Considers only those reactions that are present in all the networks present below the current level.
  3. Reaction Neighbourhood: Considers reactions which meet the neighbourhood cutoff, i.e. the proportion of neighbours present in at least one of the two networks below the current level is greater than the specified cutoff.
  4. Parsimony: Builds networks by maximum parsimony approach using the Fitch algorithm (Fitch 1971) with ties resolved in the favor of presence of a reaction.
  Output Type Two types of output can be generated.
  1. Tabular: Generates a tab delimited output. Allows exporting into Microsoft Excel or other spreadsheet solution.
  2. Descriptive: Results are grouped into headings.
  Output Format Output can be generated in two formats.
  1. Text: Plain text output with no hot-links.
  2. HTML: The entities (organisms, reactions, metabolites etc.) are hot-linked to respective entries in KEGG.
